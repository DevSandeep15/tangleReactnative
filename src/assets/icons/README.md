# Tab Icons

To add custom icons for your bottom tabs, place your icon images in this folder with the following names:

- `home.png` - Home tab icon
- `service.png` - Service tab icon
- `add.png` - Add tab icon
- `hub.png` - Hub tab icon
- `profile.png` - Profile tab icon

**Recommended specifications:**
- Size: 24x24 or 48x48 pixels (for @2x)
- Format: PNG with transparency
- Color: Black or single color (will be tinted based on active/inactive state)

You can also create @2x and @3x versions for different screen densities:
- `home@2x.png` (48x48)
- `home@3x.png` (72x72)
